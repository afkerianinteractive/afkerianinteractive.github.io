2. WEBSITE OPERATION

The Website is a static GitHub Pages site. The inspected Website contains no first-party account, registration, payment, comment, subscription, contact form, advertising, or analytics system. It does not set first-party analytics or advertising cookies and does not use browser storage for tracking.

Website scripts provide local navigation, visual effects, and same-origin display of legal TXT documents. Those legal-document requests are ordinary Website requests and are not used by us to create a visitor profile.
